# A Materials List Generation Website

This is a website that can generate materials list, Makers inputs whatever he wants to make such as robotic arms and RC cars, AI will give the Maker some suggestions and generate a materials list which can be download as .CSV file.

## Quick Set Up

Download this repo and open the cmd, cd to the location where the file "tfm" are located.

Type these code to run the server.
```
npm run dev
```

After the server starts, there will be a link to the website which lookes like
```
http://localhost:5173/
```

## Development Environment

Using VSCode for coding, use VITE to set the server up, use the REACT architecture.
